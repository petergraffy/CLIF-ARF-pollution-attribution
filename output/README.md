## Output directory

There are no intermediate results for this project, only final.

After running `01_cohort_identification.R`, upload the entire `output/final/` folder.

The site-facing masked export files are:
- `site_county_year_*.csv`
- `site_county_year_age_sex_*.csv`
- `site_year_age_sex_race_ethnicity_*.csv`
- `site_qc_summary_*.csv`

Additional subfolders such as `consort/` and `table1/` are also part of the final deliverable and should be uploaded as generated.
